Lambda Calculus is an extremely minimalistic programming language, where everything is represented as a function. Functions in Lambda Calculus will always take some other function as an input, and return another as output. The simplest possible function in Lambda Calculus is the Identity Function:
(\x.x)
Inputs are denoted with the Lambda symbol (Represented as "\" in code), and seperated by ".". Everything else within the parentheses is an output. Here's how you can evaluate this function:
(\x.x) y -> (\y.y) -> y
The first step is to pass y as an input. This is denoted by writing y to the right of the function body. After that, we substitute the first lambda value with the first input throughout. Finally, we have the function (\y.y), which is identical to just y. (\x.x) is called the Identity Function, as it returns whatever it receives as input.
